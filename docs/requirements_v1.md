# Requirements V1 (Frozen)

We are building a mobile idle + dungeon ladder game.

## Core
- Cozy, space-themed idle/base game with async online cloud save
- Multiple bases (Earth, Moon, Asteroid), parallel production with storage caps
- Continuous idle production per second; claim-all per base
- Offline progress: server-authoritative, 12h cap, grants instantly on login
- No leaderboards, trading, clans, ads, or IAPs in V1 (but design future-ready)

## Skills
- Production skills: Mining, Farming, Ranching/Biofarming, Refining, Crafting, Tech
- Combat skills: Attack, Defense, Health — all level via dungeon play
- XP: idle = baseline; dungeons = burst XP

## Dungeons
- Tiered dungeon ladder (25 tiers): must clear highest to unlock next; replay allowed
- Dungeon runs (2–5 min): branching choices affect skills used (XP), rewards mostly choice-agnostic
- Auto-resolved combat + small skill-based mechanic that sets a 1×–3× multiplier
- Partial rewards on failure; no punitive penalties
- Dungeons grant rare items (incl. ship parts) + credits/resources

## Progression gating
- Ship upgrades + ship parts gate base/location progression (Europa = end of early game target)
